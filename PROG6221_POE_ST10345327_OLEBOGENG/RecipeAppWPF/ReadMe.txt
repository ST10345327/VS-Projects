 # Recipe App WPF

This application allows users to enter and store multiple recipes, each with ingredients and steps, using a graphical user interface built with WPF. 
It also calculates the total calories of each recipe and warns the user if the total calories exceed 300. Users can filter recipes by ingredient, food group, or calorie count.

## Features

- Enter multiple recipes with names.
- Display recipes in alphabetical order.
- Add ingredients with name, quantity, unit, calories, and food group.
- Calculate and display total calories of a recipe.
- Notify the user if the total calories exceed 300 using a delegate.
- Filter recipes by ingredient, food group, or maximum calories.

## How to Run

1. Clone the repository from GitHub.
2. Open the solution in Visual Studio.
3. Build the solution to restore dependencies.
4. Run the `RecipeAppWPF` project.

## How to Run Unit Tests

1. Open the solution in Visual Studio.
2. Build the solution to restore dependencies.
3. Open the Test Explorer (Test > Test Explorer).
4. Run all tests to ensure everything works correctly.
